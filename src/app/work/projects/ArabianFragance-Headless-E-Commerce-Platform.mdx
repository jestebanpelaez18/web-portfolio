## Overview

Arabian Fragrance is a luxury perfume brand launching in Helsinki 
with a catalogue of 20+ products including perfumes, body mists, 
and traditional oud blends. I designed and built the full e-commerce 
platform from scratch as a freelance project, handling everything 
from UI design to deployment.

The platform uses a headless architecture — a custom Next.js frontend 
connected to Shopify as the backend commerce engine via the Storefront 
API.

## Key Features

- Designed and built the full storefront with Next.js, TypeScript, 
  and Tailwind CSS, covering landing page, collections, product pages, 
  shop, and about sections.
- Integrated the Shopify Storefront API via GraphQL to manage product 
  data, cart logic, inventory, and dynamic rendering from a headless 
  setup.
- Built reusable component architecture for consistent UI across 
  all pages and product categories.
- Configured the Shopify store including product catalogue, 
  collections, and store settings.
- Implemented cross-browser and responsive testing with Playwright 
  to ensure consistent behavior across browsers and devices.
- Managed deployments, environment configuration, and CI/CD workflows 
  using GitHub Actions to maintain a stable live system.

## Technologies Used

- Next.js
- TypeScript
- Tailwind CSS
- Shopify Storefront API
- GraphQL
- GitHub Actions
- Playwright
- CI/CD

## Challenges and Learnings

The main challenge was designing a headless architecture that feels 
seamless to the end user while keeping the frontend fully decoupled 
from Shopify. Managing cart state across a stateless Next.js app and 
keeping the GraphQL queries efficient required careful design of the 
data layer.

## Outcome

A live production e-commerce platform for a luxury fragrance brand, 
fully custom-built and deployed, handling product browsing, cart 
management, and checkout through a headless Shopify integration.
