## Overview

Transcendence is a real-time multiplayer Pong platform developed by a team of five using an Agile workflow. I led backend development end to end, designing and implementing the core services that power authentication, matchmaking, and gameplay state.

## Key Features

- Designed and implemented REST APIs with Django and PostgreSQL for user lifecycle, matchmaking, and persistent game data.
- Implemented WebSocket communication for low-latency real-time game events and player synchronization.
- Containerized backend services with Docker and served production traffic behind Nginx.
- Built and executed automated API tests with Postman to validate authentication, matchmaking, and gameplay endpoints.

## Technologies Used

- Django
- PostgreSQL
- WebSockets
- Docker
- Nginx
- Postman

## Challenges and Learnings

The hardest part was keeping gameplay responsive while ensuring server authority and stable matchmaking. This required careful event design for WebSocket messages, predictable backend state transitions, and disciplined API validation to reduce integration regressions.

## Outcome

The platform shipped as a complete multiplayer product with reliable backend flows, stable real-time interactions, and stronger release confidence through automated test coverage.
