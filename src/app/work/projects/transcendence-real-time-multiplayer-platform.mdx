## Overview

Transcendence is a real-time multiplayer Pong platform built by a 
team of five using an Agile workflow. I owned backend development end 
to end, designing and implementing the core services that power 
authentication, matchmaking, gameplay state, and tournament management.

## Key Features

- Designed and implemented REST APIs with Django and PostgreSQL for 
  user lifecycle, matchmaking, persistent game data, and tournament 
  management.
- Implemented JWT-based authentication for secure token-based user 
  sessions including registration, login, and profile management.
- Built WebSocket communication using Django Channels for low-latency 
  real-time game events and player synchronization.
- Implemented server-side Pong logic handling ball movement, scoring, 
  and player interactions, supporting both web interface and CLI 
  interaction.
- Implemented GDPR compliance features including user anonymization 
  and local data management to meet regulatory requirements.
- Containerized all backend services with Docker and served production 
  traffic behind Nginx.
- Built and executed automated API tests with Postman to validate 
  authentication, matchmaking, and gameplay endpoints.

## Technologies Used

- Django

## Challenges and Learnings

The hardest part was keeping gameplay responsive while ensuring server 
authority over game state and stable matchmaking. This required careful 
WebSocket event design, predictable backend state transitions, and 
disciplined API validation to reduce integration regressions across 
a team of five working on different layers of the stack.

## Outcome

The platform shipped as a complete multiplayer product with reliable 
backend flows, stable real-time interactions, tournament support, and 
stronger release confidence through automated test coverage.
